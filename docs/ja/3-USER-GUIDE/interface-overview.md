# インターフェース概要

## 画面構成

### サイドバー
- ノートブック一覧
- クイックアクション（新規作成）
- テーマ切替
- 言語設定

### メインエリア
- ソース一覧 / 検索結果
- ノートエディタ
- チャットインターフェース

### 設定画面
- AIプロバイダー設定（Settings → API Keys）
- モデル選択
- デフォルトモデル設定

## キーボードショートカット

| キー | アクション |
|------|-----------|
| Ctrl+K | クイックアクション |
| Ctrl+Enter | チャット送信 |
| Esc | モーダルを閉じる |
