# ノートブック・ソース・ノート

## 概要

Open Notebook のデータは3層構造で整理されます：

### 1. ノートブック（Notebook）
最上位の整理単位です。プロジェクトやテーマごとに作成します。
- 例：「機械学習研究」「製品企画」「読書ノート」

### 2. ソース（Source）
ノートブック内の情報源です。以下の形式に対応：
- 📄 PDF / ドキュメント
- 🎥 動画（YouTube等）
- 🎵 音声ファイル
- 🌐 Webページ
- 📝 テキスト / Markdown

各ソースはAIによって自動処理（文字起こし・埋め込み生成）されます。

### 3. ノート（Note）
ソースから得た知見を自由形式で保存します。
- 手動で書くことも、AIに生成させることも可能
- Markdown形式対応
- ソースにリンク可能

## 関係図

`
ノートブック
├── ソースA（PDF）
│   ├── ノート1（要点）
│   └── ノート2（考察）
├── ソースB（YouTube動画）
│   └── ノート3（文字起こしからの抜粋）
└── ソースC（Webページ）
`
