Shared utility modules used across the library and API. No business logic; pure helpers and cross-cutting concerns.

## Summary

- chunking.py — text chunking strategies for embedding and retrieval pipelines
- embedding.py — multi-provider embedding generation helpers
- context_builder.py — assembles RAG context strings from retrieved chunks for LLM prompts
- token_utils.py — token counting and truncation helpers
- text_utils.py — general text cleaning and normalization
- encryption.py — symmetric encryption/decryption for credential storage
- error_classifier.py — `classify_error()` maps raw provider exceptions to typed `OpenNotebookError` subclasses
- graph_utils.py — shared helpers for LangGraph node construction
- version_utils.py — reads the package version at runtime

## Constraints

- MUST NOT import from `api/`; utils are a dependency layer below the API.
