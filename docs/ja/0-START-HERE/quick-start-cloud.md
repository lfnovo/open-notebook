# クイックスタート：クラウド

## 概要

クラウド環境（AWS, GCP, Azure, VPS等）でのセットアップ手順です。

## 手順

1. クラウドインスタンスを作成（Ubuntu 22.04推奨）
2. Docker と Docker Compose をインストール
3. [Docker Composeインストール](../1-INSTALLATION/docker-compose.md) の手順に従う
4. 必要に応じて[リバースプロキシ](../5-CONFIGURATION/reverse-proxy.md)を設定

> 原文: [Quick Start Cloud](../quick-start-cloud.md)
