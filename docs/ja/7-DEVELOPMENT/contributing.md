# コントリビューション（貢献）ガイド

## 貢献方法

1. リポジトリをフォーク
2. フィーチャーブランチを作成
3. 変更を実装
4. テストを実行
5. プルリクエストを作成

## 開発環境

[開発環境セットアップ](development-setup.md) を参照してください。

## コード規約

- Python: [コード規約](code-standards.md) に従う
- TypeScript/React: ESLint + Prettier

## コミットメッセージ規約

`
type(scope): description

例:
feat(i18n): add Japanese translations
fix(api): resolve search timeout
docs: update installation guide
`

## プルリクエストの前に

- [ ] テストが通ること
- [ ] コード規約に準拠していること
- [ ] 変更内容をCHANGELOGに反映（該当する場合）
