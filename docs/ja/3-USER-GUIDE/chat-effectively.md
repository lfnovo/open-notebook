# 効果的なチャット

## チャットのベストプラクティス

### 1. コンテキストを絞る

全ソースを一度に使うより、関連するソースだけを「全文」モードに設定すると精度が上がります。

### 2. 具体的な質問をする

| ❌ 悪い例 | ✅ 良い例 |
|----------|---------|
| 「これについて教えて」 | 「この論文の実験結果を要約して」 |
| 「どう思う？」 | 「このデータのトレンドを分析して」 |

### 3. チャットセッションを使い分ける

- ノートブック全体チャット → プロジェクト全体の質問
- ソース個別チャット → 特定のドキュメント深掘り

### 4. 変換（Transformation）を活用

定型処理は変換を使うと効率的：
- 要約
- キーポイント抽出
- 用語解説
- 翻訳
