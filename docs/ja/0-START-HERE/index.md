# はじめに

Open Notebook を始めるには、使用するAIプロバイダーに応じたクイックスタートガイドを選んでください。

## クイックスタートガイド

| ガイド | 対象 |
|--------|------|
| [OpenAI クイックスタート](quick-start-openai.md) | OpenAI APIキーをお持ちの方 |
| [クラウド クイックスタート](quick-start-cloud.md) | クラウド環境でのセットアップ |
| [ローカル クイックスタート](quick-start-local.md) | Ollamaを使った完全ローカルセットアップ |
| [外部Ollama クイックスタート](quick-start-external-ollama.md) | 別マシンのOllamaサーバーに接続 |

## インストール方法

詳細なインストール手順は[インストールガイド](../1-INSTALLATION/index.md)を参照してください。

| 方法 | 説明 |
|------|------|
| [Docker Compose](../1-INSTALLATION/docker-compose.md) | 推奨。最も簡単な方法 |
| [ソースから](../1-INSTALLATION/from-source.md) | 開発者向け |
| [シングルコンテナ](../1-INSTALLATION/single-container.md) | 最小構成 |
| [Windowsネイティブ](../1-INSTALLATION/windows-native.md) | Windows上で直接実行 |
