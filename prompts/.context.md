Jinja2 prompt templates for all AI workflows. Templates are loaded by `ai-prompter` and rendered at runtime with graph state variables.

## Summary

- ask/ — templates for the ask (Q&A) graph: entry prompt, query processing, final answer synthesis
- chat/ — system prompt template for the notebook chat graph
- podcast/ — templates for podcast outline generation and full transcript creation
- source_chat/ — system prompt template for single-source chat graph

## Guidance

- SHOULD keep templates focused on structure and tone, not hardcoded content; use template variables for runtime data.
