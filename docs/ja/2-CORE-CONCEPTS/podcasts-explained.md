# ポッドキャスト解説

## 概要

Open Notebook は最大4人の話者によるAIポッドキャストを生成できます。

### 主な機能

- **マルチスピーカー**: 1〜4人の話者を設定可能
- **話者プロファイル**: 各話者の名前・声質・話し方をカスタマイズ
- **エピソードプロファイル**: ポッドキャストの形式・長さ・トーンを設定
- **スクリプト制御**: AIが生成したスクリプトを編集可能
- **音声合成（TTS）**: 複数のTTSプロバイダーに対応（OpenAI TTS, ElevenLabs 等）

### ワークフロー

1. エピソードプロファイルを作成（トピック、長さ、スタイル）
2. 話者プロファイルを割り当て
3. ソースを選択（ポッドキャストの材料）
4. 生成実行
5. スクリプトを確認・編集
6. 音声を生成・ダウンロード
