# トラブルシューティング：AIチャットの問題

## AIが応答しない

1. APIキーが正しいか確認（Settings → API Keys → 接続テスト）
2. モデルが同期されているか確認（Settings → API Keys → モデル同期）
3. クレジット残高を確認（OpenAI等のプロバイダーダッシュボードで）

## 応答が遅い

- Ollama使用時：より小さいモデルを試す
- ネットワーク接続を確認
- `ESPERANTO_LLM_TIMEOUT` を増やす（`.env` で設定）

## 回答の質が低い

- 質問をより具体的にする
- ソースの品質を確認（テキスト抽出が正確か）
- より高性能なモデルに切り替える
