# AIコンテキストとRAG（Retrieval-Augmented Generation）

## RAGとは

RAG（検索拡張生成）は、AIがあなたのデータに基づいて回答を生成する仕組みです。

### 仕組み

1. **埋め込み（Embedding）**: すべてのソースがAIモデルによってベクトル化される
2. **検索**: 質問に関連する部分をベクトル検索で発見
3. **生成**: 見つかったコンテキストをAIへのプロンプトに追加し、正確な回答を生成

`
質問 → ベクトル検索 → 関連チャンク抽出 → AIプロンプトに追加 → 回答生成
`

### コンテキストモード

各ソースには3つのコンテキストモードがあります：

| モード | 説明 |
|--------|------|
| 含めない | チャットに使用しない |
| インサイトのみ含める | AIが抽出した要約のみ使用 |
| 全文 | ソース全体をコンテキストとして使用 |
