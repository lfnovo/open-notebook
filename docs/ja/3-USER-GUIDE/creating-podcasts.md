# ポッドキャスト作成

## 手順

### 1. 話者プロファイルを作成
- Settings → Speaker Profiles
- 名前、声質（TTS音声）を設定

### 2. エピソードプロファイルを作成
- Podcasts → Episode Profiles → 新規作成
- トピック、長さ、スタイル、話者を設定

### 3. ソースを選択
- ポッドキャストの材料となるソースを選択

### 4. 生成
- 「ポッドキャストを生成」をクリック
- スクリプトが生成されるのを待つ

### 5. 確認・編集
- 生成されたスクリプトを確認
- 必要に応じて編集

### 6. 音声生成
- スクリプトから音声を生成
- ダウンロードまたは再生
