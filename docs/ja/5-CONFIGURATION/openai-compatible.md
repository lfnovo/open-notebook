# OpenAI互換API設定

OpenAI互換API（ローカルLLMサーバー等）をOpen Notebookに接続する方法です。

## 設定手順

1. Settings → API Keys → OpenAI Compatible
2. 設定を追加
3. ベースURLを入力（例：`http://localhost:1234/v1`）
4. APIキー（必要な場合）
5. モデル同期を実行

### 対応サーバー例

- LM Studio
- llama.cpp
- vLLM
- LocalAI
- text-generation-webui
