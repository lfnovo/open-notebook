Surreal-Commands async job handler definitions. These are the worker-side implementations of background jobs submitted by the API (podcast generation, embedding, source processing).

## Summary

- embedding_commands.py — job handler that triggers embedding generation for a source or note
- podcast_commands.py — job handler that runs full podcast episode generation (script → TTS → audio assembly)
- source_commands.py — job handler for async source content extraction and ingestion
- example_commands.py — reference/template command for adding new job types

## Constraints

- MUST NOT block the API process; these handlers run in a separate worker process.
