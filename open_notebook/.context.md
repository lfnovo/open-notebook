Core Python library containing domain models, AI provider integration, LangGraph workflow graphs, database access, utilities, and podcast support. The API and commands packages depend on this package for all business logic.

## Summary

- __init__.py — package init
- config.py — environment-driven application configuration (SurrealDB URL, auth settings, feature flags)
- exceptions.py — exception hierarchy rooted at `OpenNotebookError` with typed subclasses
- domain/ — Pydantic domain models and repository functions for all core entities
- ai/ — AI provider abstraction (ModelManager, Esperanto integration, credential-based provisioning)
- graphs/ — LangGraph state machine workflows (source ingestion, chat, ask, transformation)
- database/ — SurrealDB async driver wrappers, migration manager, repository base
- podcasts/ — Podcast and episode domain models and migration helpers
- utils/ — Shared utilities (chunking, embedding, encryption, context building, token counting, error classification)

## Constraints

- MUST NOT import from `api/` — this package is a dependency of the API, not the reverse.

## Guidance

- SHOULD use `provision_langchain_model()` from `ai/provision.py` for all LangGraph model selection.
- SHOULD define custom exceptions in `exceptions.py`; never raise raw `Exception` from library code.
