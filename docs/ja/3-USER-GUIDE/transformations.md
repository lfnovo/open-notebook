# 変換（Transformations）

## 概要

変換は、AIを使ったコンテンツの定型処理です。チャットとは異なり、プロンプトテンプレートを使って一貫した結果を得られます。

### ビルトイン変換

| 変換 | 説明 |
|------|------|
| 要約 | ソースの簡潔な要約を生成 |
| キーポイント | 重要なポイントを箇条書きで抽出 |
| 詳細分析 | 深い分析と考察 |
| 用語集 | 専門用語の定義リスト |

### カスタム変換

独自のプロンプトテンプレートを作成して、任意の変換処理を定義できます。

`
例: 「以下の内容を日本語で3行以内に要約してください」
`

### 一括モード

複数ソースに対して一括で変換を実行可能。
