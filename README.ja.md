## プライベート・マルチモデル・100%ローカル対応の Notebook LM 完全代替

![New Notebook](docs/assets/asset_list.png)

人工知能が世界を変える中、🧠考え 💡新しい知識を得る力は、一部の特権でも単一プロバイダーに制限されるべきものでもありません。

**Open Notebook でできること：**
- 🔒 **データ管理** - リサーチを非公開・安全に保つ
- 🤖 **AIモデルの自由選択** - OpenAI, Anthropic, Ollama, LM Studio など18以上のプロバイダー対応
- 📚 **マルチモーダル整理** - PDF、動画、音声、Webページなど多様な形式に対応
- 🎙️ **プロ品質のポッドキャスト** - 高度なマルチスピーカー生成
- 🔍 **インテリジェント検索** - 全文検索＋ベクトル検索のハイブリッド
- 💬 **コンテキスト付きチャット** - 自分のリサーチを活用したAI対話
- 🌐 **多言語UI** - 日本語、英語、ポルトガル語、中国語（簡体・繁体）、ロシア語、ベンガル語

プロジェクトの詳細は [https://www.open-notebook.ai](https://www.open-notebook.ai) をご覧ください。

## 🆚 Open Notebook vs Google Notebook LM

| 機能 | Open Notebook | Google Notebook LM | 優位点 |
|---------|---------------|--------------------|-----------|
| **プライバシーと管理** | セルフホスト、データは手元で管理 | Googleクラウドのみ | 完全なデータ主権 |
| **AIプロバイダー選択** | 18以上のプロバイダー（OpenAI, Anthropic, Ollama, LM Studio等） | Googleモデルのみ | 柔軟性とコスト最適化 |
| **ポッドキャスト話者数** | 1〜4人、カスタムプロファイル付き | 2人のみ | 圧倒的な柔軟性 |
| **コンテンツ変換** | カスタム＋ビルトイン | 限定的 | 無限の処理能力 |
| **APIアクセス** | 完全なREST API | APIなし | 完全自動化 |
| **デプロイ** | Docker、クラウド、ローカル | Googleホストのみ | どこでも展開可能 |
| **引用** | 基本参照（改善予定） | ソース付き包括的 | 研究の整合性 |
| **カスタマイズ性** | オープンソース、完全カスタマイズ可 | クローズドシステム | 無限の拡張性 |
| **コスト** | AI使用料のみ | 無料枠＋月額サブスク | 透明で管理しやすい |

**Open Notebook を選ぶ理由：**
- 🔒 **プライバシー最優先**: 機密リサーチを完全に非公開に
- 💰 **コスト管理**: 安価なAIプロバイダーを選ぶか、Ollamaでローカル実行
- 🎙️ **より良いポッドキャスト**: 完全なスクリプト制御とマルチスピーカー対応
- 🔧 **無制限のカスタマイズ**: 必要に応じて修正・拡張・統合
- 🌐 **ベンダーロックインなし**: プロバイダー切替自由、データは自分のもの
