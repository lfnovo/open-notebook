# 検索

## ハイブリッド検索

Open Notebook は**全文検索**と**ベクトル検索**のハイブリッド方式です。

### 全文検索
- キーワードベースの高速検索
- ソースタイトル、ノート内容を対象

### ベクトル検索（セマンティック検索）
- 意味ベースの検索
- 類義語や言い換えにも対応
- ソースの埋め込み（Embedding）データを使用

### 質問（Ask）モード
検索との違い：
- **検索**: 関連するドキュメント部分を見つける
- **質問**: AIがソースに基づいて回答を生成する

`
POST /api/search         → 検索結果
POST /api/search/ask     → AIによる回答（RAG）
`
