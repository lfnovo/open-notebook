# ソースの追加

## 対応形式

| 形式 | 方法 | 処理内容 |
|------|------|---------|
| PDF | アップロード | テキスト抽出 → 埋め込み生成 |
| YouTube | URL入力 | 字幕取得 → 埋め込み生成 |
| Webページ | URL入力 | 本文抽出 → 埋め込み生成 |
| 音声/動画 | アップロード | 文字起こし（STT）→ 埋め込み生成 |
| テキスト | 直接入力 | 埋め込み生成 |
| Office文書 | アップロード | テキスト抽出 → 埋め込み生成 |

## ソース追加手順

1. ノートブックを開く
2. 「新規ソース」をクリック
3. ソースタイプを選択
4. ファイルをアップロード または URLを入力
5. 処理完了を待つ（数秒〜数分）

### 処理ステータス

- 🔄 **処理中**: AIが解析中
- ✅ **完了**: 利用可能
- ❌ **エラー**: 再試行可能
