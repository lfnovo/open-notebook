# チャット vs 変換（Transformations）

## 違い

| 観点 | チャット | 変換 |
|------|---------|------|
| 目的 | 対話・探索 | 定型処理 |
| プロンプト | 自由入力 | テンプレート |
| 結果 | 会話履歴付き | 単一結果 |
| 再利用 | 低い | 高い |

## 使い分け

- **チャット**: 「この論文の主張を批判的に検討して」
- **変換**: 「すべてのソースを3行で要約」
